# Advanced HTML & CSS Project - My First Portfolio

## Project Overview
This project is part of the HTML & CSS advanced course. The goal was to build a fully responsive landing page based on a Figma design.

## Features
- Fully responsive layout for desktop, tablet, and mobile
- Flexbox-based layout for header, hero section, tutorials, and footer
- Styled sections: Hero, Learn from the Pros, Testimonials, Tutorials, Free Membership, FAQ
- Buttons, cards, stars, and icons styled using CSS and Font Awesome
- Viewport meta tag added for proper scaling on mobile devices

## Technologies Used
- HTML5
- CSS3 (Flexbox, responsive media queries)
- Font Awesome icons
- Figma (design reference)

## File Structure

## How to Run
1. Open `index.html` in any modern browser.
2. Resize the window to see the responsive design in action.
3. All CSS rules are included in `styles.css`.
